# MonRix - Trade Risk Monitoring Platform

## Overview

MonRix is a Django-based Trade Risk Monitoring Platform designed to help portfolio managers track trades, monitor portfolio exposure, calculate unrealized profit and loss, and enforce risk limits.

The project was built to demonstrate backend development skills using Django, Django ORM, PostgreSQL, Django REST Framework, and authentication systems.

## Features

### Authentication

* User Registration
* Login and Logout
* Portfolio ownership and access control

### Portfolio Management

* Create portfolios
* View portfolio details
* Risk limit configuration

### Trade Management

* Buy and Sell trade recording
* Trade history tracking
* Symbol filtering and search

### Portfolio Analytics

* Net Position Tracking
* Exposure Calculation
* Average Cost Basis
* Unrealized P&L

### Risk Monitoring

* Portfolio Risk Limits
* Risk Breach Alerts

### APIs

* Portfolio API
* Trade API
